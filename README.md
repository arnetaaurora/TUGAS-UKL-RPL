## DESKRIPSI 
Repository ini berisi kumpulan program java untuk latihan (UKL) setiap file berisi program dengan fungsi yang berbeda-beda seperti perhitungan matematika , logika , dan struktur perulangan 
## DAFTAR PROGRAM 
1.BiayaPengiriman.java (menghitung total biaya pengiriman barang) 
2.GanjilGenap.java (menentukan apakah angka termasuk ganjil atau genap)
3.HitungFaktorial.java (menghitung faktorial dari suatu angka)
4.Volumetabung (menghitung volume tabung dari jari-jari dan tinggi) 
5.RataRataNilai.java(menghitung rata-rata dari bebera nilai)
6.PenjumlahanMatriks.java(menjumlahkan dua buah matriks)
## CARA MENJALANKAN PROGRAM 
1.Buka file java yang ingin dijalankan 
2.Jalankan program di VS CODE 
3.Masukkan input sesuai intruksi , lalu lihat hasil outputnya
